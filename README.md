# Plangent

Plangent — десктопное приложение для оркестрации AI-агентов, работающих с
вашими репозиториями: вы создаёте проект, ставите задачу, и Plangent проводит
агента (или очередь агентов) через цикл выполнения по плану — со встроенным
терминалом, историей запусков и настройкой агентов/скиллов.

## Установка

Скачайте актуальную сборку для своей ОС со страницы
[Releases](https://github.com/BlPavel/Plangent/releases/latest).

### macOS

1. Скачайте `Plangent-<версия>.dmg`.
2. Откройте `.dmg` и перетащите **Plangent** в папку `Applications`.
3. Сборка не подписана и не нотаризована Apple, поэтому при первом запуске
   Gatekeeper заблокирует приложение. Кликните правой кнопкой (или
   Control+клик) по **Plangent** в `Applications`, выберите **Открыть** и
   подтвердите **Открыть** в диалоге. Это нужно сделать только один раз.

### Windows

1. Скачайте `Plangent-Setup-<версия>.exe`.
2. Запустите установщик. Так как сборка не подписана сертификатом,
   Windows SmartScreen покажет предупреждение «Windows защитила ваш
   компьютер» — нажмите **Подробнее**, затем **Всё равно запустить**.
3. Следуйте шагам установщика, при желании изменив папку установки.
